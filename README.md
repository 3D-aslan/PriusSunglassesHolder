# Prius Gen 5 – Sunglasses Holder (3D Printed)

This is a custom sunglasses holder I designed for the 2023–2024 Toyota Prius.  
The car doesn’t have a good storage spot for sunglasses, so I made a part that fits securely in the center area and holds the glasses without scratching them.
